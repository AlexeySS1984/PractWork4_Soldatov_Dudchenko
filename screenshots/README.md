## 🚀 Как создать скриншоты:

**Подробная инструкция находится в файле:** `SCREENSHOTS_GUIDE.md` в корне проекта

### Кратко:

1. **Запустите приложение** (F5)
2. **Для каждой страницы:**
   - Введите тестовые данные
   - Нажмите "Вычислить"
   - Сделайте скриншот (Win+Shift+S)
   - Сохраните в эту папку
3. **Для Test Explorer:**
   - Test → Test Explorer
   - Run All (запустить все тесты)
   - Сделайте скриншот всех 36 пройденных тестов

---

## ✅ После создания скриншотов:

1. Замените placeholder файлы на реальные скриншоты
2. Убедитесь, что имена файлов совпадают точно
3. Добавьте в Git: `git add screenshots/`
4. Закоммитьте: `git commit -m "Добавлены скриншоты"`
5. Запушьте: `git push`

---

## 📏 Требования к скриншотам:

- ✅ Формат: PNG
- ✅ Размер: 800-1920px по ширине
- ✅ Содержимое: Чёткое и читаемое
- ✅ Весь интерфейс должен помещаться в кадр
