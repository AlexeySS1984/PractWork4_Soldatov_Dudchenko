**Ограничение:** y > 0

### Функция E (Страница 2)
**Кусочно-заданная функция:**
- `i√f(x)`, если i - нечётное, x > 0
- `i/2·√|f(x)|`, если i - чётное, x < 0

где `f(x) ∈ {sh(x), x², e^x}`

**Формула:** `y = x⁴ + cos(2 + x³ - b)`
