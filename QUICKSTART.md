## ⚡ Как начать работу:

### Шаг 1: Открыть проект
1. Распакуйте архив `PractWork4_Soldatov_Dudchenko.zip`
2. Откройте Visual Studio 2019 или новее
3. File → Open → Project/Solution
4. Выберите файл `PractWork4_Soldatov_Dudchenko.csproj`

### Шаг 2: Восстановить пакеты
1. В Visual Studio: Tools → NuGet Package Manager → Manage NuGet Packages for Solution
2. Нажмите "Restore" если появится уведомление
3. Пакет LiveCharts.Wpf будет установлен автоматически

### Шаг 3: Запустить приложение
1. Нажмите F5 или Debug → Start Debugging
2. Приложение откроется с первой страницей
3. Протестируйте все три страницы

### Шаг 4: Создать Git репозиторий
1. Откройте файл `GIT_INSTRUCTIONS.md`
2. Следуйте инструкциям по созданию репозитория
3. Добавьте преподавателя TGAksenova как соразработчика
4. Отправьте первый коммит

## 🆘 Возникли проблемы?

### Ошибка: "LiveCharts не найден"
→ Решение: Tools → NuGet Package Manager → Restore Packages

### Ошибка: "Изображения не отображаются"
→ Решение: Проверьте, что файлы formula*.png в корне проекта

### Ошибка при Push в Git
→ Решение: Сначала сделайте Pull, затем снова Push

### Конфликт слияния в Git
→ Решение: Откройте файл, разрешите конфликт вручную
