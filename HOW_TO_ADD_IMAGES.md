## 📸 Как заменить изображения (необязательно):

### Вариант 1: Скриншот из PDF задания

1. Откройте файл с заданием (PDF)
2. Найдите нужную формулу для вашего варианта
3. Сделайте скриншот формулы (Windows: Win + Shift + S)
4. Сохраните изображение с соответствующим названием

### Вариант 2: Использование онлайн редакторов LaTeX

1. Перейдите на сайт: https://www.codecogs.com/latex/eqneditor.php
2. Введите формулу в формате LaTeX
3. Скачайте изображение
4. Переименуйте файл

### Вариант 3: Microsoft Word

1. Вставьте формулу в Word используя редактор формул
2. Сделайте скриншот формулы
3. Обрежьте изображение
4. Сохраните с нужным именем

## Как добавить изображения в проект:

1. Скопируйте файлы изображений в папку проекта
2. В Visual Studio: правой кнопкой на проекте → Add → Existing Item
3. Выберите все три изображения
4. В свойствах каждого изображения установите:
   - Build Action: Content
   - Copy to Output Directory: Copy if newer

## Размер изображений:

Рекомендуемые размеры:
- Ширина: 400-600 пикселей
- Высота: зависит от формулы (обычно 60-120 пикселей)
- Формат: PNG (с прозрачным фоном желательно)
- Разрешение: 96 DPI или выше

## Если изображения не отображаются:

1. Проверьте, что файлы находятся в корневой папке проекта
2. Проверьте названия файлов (должны совпадать точно)
3. Проверьте, что в .csproj есть правильная настройка копирования
4. Пересоберите проект (Build → Rebuild Solution)
